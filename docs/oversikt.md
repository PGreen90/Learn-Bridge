# Budsystem – översikt (snabbreferens)

> Kondenserad översikt av vårt 2-över-1-system. Fullständig detalj finns i
> [budsystem.md](budsystem.md). Förkortningar: **hf** = högfärg (♥/♠),
> **lf** = lågfärg (♣/♦), **hp** = honnörspoäng, **GF** = utgångskrav.

## Öppningar
| Bud | Betydelse |
|---|---|
| 1♣ / 1♦ | 3+, 12+ hp (minor-regeln) |
| 1♥ / 1♠ | 5+, 12+ hp |
| 1NT | 15–17 balanserad |
| 2♣ | 22+ / stark, konstgjord (krav) |
| 2♦ / 2♥ / 2♠ | svaga tvåor, 6-korts, 6–11 hp |
| 2NT | 20–21 balanserad |
| 3-läget / 4-läget | spärr · 3NT 25–27 |

## Svar på 1♥ / 1♠
- Ny färg 1-steg · semi-forcing 1NT · 2/1 GF
- **Bergen:** 3♣ = 7–10 · 3♦ = 10–12 · 3 i hf = spärr
- Enkel höjning + **Bergen game try** (2NT)
- **Jacoby 2NT** (4+ stöd, 13+) · **tvetydig splinter** (3♠/3♥)
- 3NT = 13–15 balanserad

## Svar på 1♣ / 1♦
- 4-korts hf upp · 1NT 6–10 (ej krav)
- **Inverterade minorer:** 1m–2m stark 10+ (krav), 1m–3m svag spärr
- 2/1 GF · 2NT 11–12 · 3NT 13–15

## Svar på 1NT
- **2♣** Stayman · **2♦→♥, 2♥→♠** Jacoby · **2♠** Minor Suit Stayman
- **Smolen** (1NT–2♣–2♦–3♥/3♠) · **Texas** 4♦/4♥
- 2NT 8–9 (inbjudan) · 3NT 10–15

## Stark 2♣
- **2♦ väntebud**; positiva svar 8+ med 5+ färg
- Öppnaren: 2NT 22–24 · 3NT 28–30 · färgrebud = krav
- **Andra negativa** (billigaste minor 0–3) · träff → cue / 1430 RKC

## Svaga tvåor & spärrar
- **Ogust** på 2NT (steg 3♣–3NT: min/max + färgkvalitet)
- Spärrsvar: ny färg krav · 4NT = 1430 RKC · regel om 2-3-4

## Öppnarens återbud
- Styrkenivåer 12–15 / 16–18 / 19+ · reverse 16+ · 2NT-rebud 18–19
- 2/1 GF: bjud lugnt ("fast arrival")

## Slamverktyg
- **1430 RKC** (5 nyckelkort) · **cue-bid** (kontroll) · **Sjöbergs 5NT** (vilken kung)
- **Gerber** (4♣ över NT) · **Exclusion** (5-läges voidwood)
- **Fjärde färg krav** (GF) · **Drury** (tvåvägs Reverse)

## Försvar & konkurrens
- Inkliv · 1NT-inkliv 15–18 (kör 1NT-systemet)
- **Michaels + ovanlig 2NT** (två zoner)
- Dubblingar: upplysning · **negativ** · **responsiv** · **stöd**
- **Lebensohl** · **DONT** mot deras 1NT · **Mathe** mot stark 1♣

## Markeringar & utspel
- **UDCA** (omvänd attityd + räkning) · **Lavinthal**-sak
- **3:e/5:e** utspel · standard honnörsutspel
